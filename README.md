# Injection Workshop

## Requirements
* Java 1.6+
* Maven
* A Java IDE

## Package & Run
* ```cd injection```
* ```mvn clean package```
* ```java -jar target/injection-1.0-SNAPSHOT-jar-with-dependencies.jar```
