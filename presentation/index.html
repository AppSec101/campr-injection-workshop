## Injection? - What are injection vulnerabilities? - What’s the worst that can happen?
## Injection - "Injection flaws occur when an application sends untrusted data to an interpreter." - SQL, LDAP, Xpath, or NoSQL queries, OS commands, XML parsers, SMTP Headers, program arguments
